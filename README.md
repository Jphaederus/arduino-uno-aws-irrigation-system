<h1>ABOUT</h1>
<p>This is an extremely popular DIY project because it automatically waters your plants and flowers according to the level of moisture in the soil.</p>
<p>The list of item above combines the development board (Arduino Uno), Mini pump, Tubing, Soil Moisture Sensors and a 4 Channel 5V Relay Module, which is all you need to build an automated irrigation system.</p>
